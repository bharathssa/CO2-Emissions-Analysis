# Project Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-14

### Added
- Initial project setup with data collection and cleaning
- CO2 emissions data import from UNFCCC inventories
- GDP and population data integration from World Bank and web scraping
- Data filtering for energy sector subsectors (1.A.1.a, 1.A.1.b, 1.A.1.c)
- Exploratory data analysis pipeline
- Correlation analysis between GDP, population, and emissions
- Time-series analysis for 101 countries (2000-2021)
- 2050 emissions forecasting models

### Features
- Comprehensive Jupyter notebook with full analysis pipeline
- Data preprocessing and cleaning workflows
- 37,829 records of processed emissions data
- Interactive visualizations using Plotly
- Statistical analysis with Pandas and NumPy
- Professional documentation and project structure

### Data
- 50,000+ rows of UN UNFCCC CO2 emissions data
- 101 countries covered
- 21-year time period (2000-2021)
- 4 primary energy subsectors analyzed
